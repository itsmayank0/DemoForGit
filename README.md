# Basic Repo For Practicing Git and Github 

# Commands Used:
`git init` 
`git stash`
`git stash list`
`git stash apply`
`git stash apply stash@{2}`
`git add -A`
`git commit -m <commit Message>`
`git branch`
`git branch -r`
`git checkout <branch>`
`git checkout --track origin/<remoteBranch>`
`git branch <new branch>`
`git push`
`git push --set-upstream origin <localbranch>`
